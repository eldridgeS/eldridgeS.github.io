VHDL FPGA LED Metronome on DE10-Standard
ECE2031 Digital Design Laboratory Group Project
My team designed a VHDL-based LED metronome peripheral for the DE10-Standard FPGA using Quartus Prime and ModelSim, implementing PWM-controlled LED brightness and clock division for adjustable BPM output. The system was structured around a modular SCOMP-based architecture to ensure accurate and scalable timing behavior. Control of tempo and LED states was enabled through SCOMP assembly, interfacing directly with memory-mapped FPGA registers. Simulation, waveform analysis, and timing closure verification were performed to ensure correct functionality and robust hardware-software integration across the entire design pipeline.

Wordle Game Embedded Systems Programming on Mbed uLCD kit
ECE2035 Programming HW/SW Systems Project
Developed a Wordle-inspired game on the Mbed uLCD platform to reinforce embedded systems programming concepts using C++ on a 32-bit ARM Cortex-M microcontroller. The project involved implementing real-time input handling via GPIO, optimizing memory usage, and synchronizing game logic with frame rendering and display updates. I integrated PWM-based sound effects and responsive graphical feedback on the uLCD screen. This project deepened my understanding of low-level hardware-software integration, timing constraints, and peripheral interfacing in resource-constrained environments.

Rewindify: A Spotify Wrapped Experience
CS2340 Objects and Design Project 2
Rewindify is a web app that offers users a fun and engaging way to explore their Spotify listening habits. It features a responsive and visually appealing interface that lets users view personalized music insights and save customized wraps by creating an account. Designed with a focus on user security and seamless interaction, Rewindify enhances the music discovery experience. FATLanta Food Finder
CS2340 Objects and Design Project 1
FATLanta Food Finder is a web application designed to help users locate restaurants in the Atlanta area. Built using the SCRUM Agile methodology, our team developed the app with Python (Django), JavaScript, HTML, and CSS. We integrated a RESTful backend API with the Google Maps API to support real-time search. The project emphasized full-stack development and team collaboration using Pycharm and GitHub, and included profile support for favorites, custom searches, and location tracking. This project strengthened our understanding of scalable web architecture and third-party API integration.

Telegram ChatBot for Artillery Maintenance
Developed a ChatBot
During my military service as an armament technician, I was responsible for maintaining a wide array of specialized tools. To improve the inventory workflow, I built a Telegram ChatBot that integrates with Google Sheets via the Telegram Bot API. The bot parses JSON requests to retrieve and update tool data, providing descriptions, images, and checklists. This streamlined tool tracking for combat exercises and enhanced technician onboarding by automating data access. The project highlights practical use of APIs, data automation, and user-focused interaction design for operational efficiency.
